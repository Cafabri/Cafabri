## 🚀 Featured Projects

**[DualSign](https://github.com/Cafabri/dual-screen-sign-translator)** — Real-time, zero-touch, bidirectional ASL ↔ English translator that runs entirely in the browser. Custom LSTM trained on the WLASL dataset (82.3% accuracy over 10 classes), on-device inference with TF.js + MediaPipe, dual-screen architecture over WebSockets, and NLP enrichment via Groq + ElevenLabs TTS.

**[StockSeer](https://github.com/C-D-Innovate/StockSeer)** — Stock prediction platform combining real-time market data with news sentiment analysis to support investment decisions. Stacking ensemble model (RandomForest + SVR + ElasticNet), Java Clean Architecture backend with ActiveMQ, and interactive Streamlit dashboard.
